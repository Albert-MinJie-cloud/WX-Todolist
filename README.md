# 微信小程序 TodoList

* 1.0 待办事项列表
    * 分类筛选(已完成、未完成、全部)
    * 排序(正序、倒序)
    * 已完成的事项不可以更新、可以查看详情、删除
    * 未完成的事项可以更新、删除、查看详情
    * 新增待办事项

* 2.0 我的日程
    * 显示日历
    * 某一日的待办事项(未有该功能)

* 3.0 时钟


* 4.0 个人中心
    * 程序的一些相关设置项
    * 登录、注册
    * 微信免密登录(首次使用需要绑定手机号)
